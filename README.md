# 📝 Task App - REST API

[![Node.js CI](https://github.com/Kenrick-W/task-app/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/Kenrick-W/task-app/actions/workflows/ci.yml)

Simple Task Management REST API built with **Node.js, Express, Jest, and GitHub Actions (CI/CD)**.

---

## 🚀 Features

- Create task (POST /tasks)
- Get all tasks (GET /tasks)
- Delete task (DELETE /tasks/:id)
- Input validation (title is required)

---

## 📦 Tech Stack

- Node.js
- Express.js
- Jest (Unit & Integration Testing)
- Supertest
- GitHub Actions (CI/CD)
